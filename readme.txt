Allows you to protect all posts from chosen post type or types.

== Description ==

Bulk Password Protect Post Types allows you to choose one or more post types and set password for all posts within.

= Features: =

* Automatically "reads" all post types on website
* Allows you to set password for all posts within selected post types

== Installation ==

1. Upload the plugin files to the "/wp-content/plugins/bulk-password" directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Click Settings or navigate to Settings -> Bulk Password Protect 
4. Choose one or more post types and password
